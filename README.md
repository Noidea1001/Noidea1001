## About

I build production-quality applications with an emphasis on clean architecture, maintainable code, and interfaces that feel finished rather than assembled. My background is backend-first — REST API design, relational data modeling, and authentication/authorization systems — and I take full ownership of frontend implementation when a project calls for it, from component structure down to responsive layout details.

My work consistently follows an API-first structure: separated repository/service layers, EF Core for data access, and a clear boundary between backend contracts and frontend consumption. I care about a codebase looking intentional, not improvised — proper folder structure, consistent naming, and documentation that lets someone else pick the project up without a walkthrough.

## Selected Projects

### WebTestPortal
Full-stack online examination platform built around a pure Web API backend consumed by a static HTML/JS frontend.

- Admin and student dashboards with distinct permission flows
- Partial-credit grading engine and question-flagging system
- Question locking once a test has been attempted, with locked-state UI
- Category system backed by a normalized lookup table (EF Core relational design)
- Analytics dashboards: score distribution, pass/fail breakdowns, mirrored across admin and student views
- Controllers organized into `Controllers/Api/` and `Controllers/Mvc/` for clear separation of concerns

`ASP.NET Core 10` · `EF Core` · `SQLite` · `Vanilla JS` · `Bootstrap 5`
[View repository →](https://github.com/Noidea1001/WebTestPortal)

### WebPlanner
Task management application centered on a kanban board and calendar view.

- Full kanban board with drag-and-drop task management
- Custom "Move to" dropdown built specifically to solve mobile drag-and-drop limitations
- Visual polish pass: pill-shaped view toggles, priority-tinted cards, column accent bars, hover interactions

`ASP.NET Core MVC` · `C#`
[View repository →](https://github.com/Noidea1001/Webplanner)

### Online Learning System
Django-based e-learning platform with AI-assisted localization.

- Neon PostgreSQL backend with Django ORM
- Google Gemini integration for English-to-Khmer auto-translation
- Custom `{% bilingual %}` template tag with CSS-based language toggling
- Migrated to full Django i18n (`LocaleMiddleware`, `.po` files) to remove client-side language flash
- Mobile responsiveness fixes across navbar, dashboard, and course action layouts

`Django` · `PostgreSQL` · `Gemini API` · `i18n`

### POS System
Point-of-sale interface project focused on clean, functional UI design.

`HTML` · `CSS`
[View repository →](https://github.com/Noidea1001/pos)
